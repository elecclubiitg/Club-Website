PROJECTS

Below here are the projects which are beign done by club. There are many projects which cover various domain of electronics.

Thermo-electronic Wilson Cloud Chamber

A project to build a Thermo Electric Wilson Chamber (TEWC) involves the design and construction of a particle detector that utilizes thermoelectric cooling and a supercooled liquid to create a cloud chamber. The goal of this project is to develop a portable and low-cost TEWC that can be used for educational and research purposes, as well as for field work in remote locations.

Class-D Audio Amplifier with Mosfet

A Class-D Audio Amplifier with Mosfet is a type of electronic circuit that amplifies audio signals to drive speakers. It uses Metal-Oxide-Semiconductor Field-Effect Transistor (MOSFET) technology to provide high efficiency and low distortion. The project which we are working on is to integrate this amplifier circuit into a media player, allowing it to play audio from various sources such as music files or streaming services. The use of a Class-D amplifier will allow for a compact and energy efficient design, while the MOSFETs provide high power output and low distortion.

Club Automation

This project aims to build smart gadgets that utilize the Internet of Things (IoT) technology to improve the convenience and efficiency of everyday life.The smart gadgets that will be developed include a range of devices such as smart thermostats, smart lighting systems, smart security cameras, and smart appliances. These devices will be connected to a central system, which can be accessed and controlled remotely using a smartphone or computer.

Fin-Flow

An IoT-based project that aims to improve water quality involving the deployment of a network of smart sensors at various locations within a water distribution system. These sensors would be able to detect and monitor various water quality parameters such as pH, temperature, turbidity, and chemical levels. The collected data would be transmitted in real-time to a central system via a wireless network, where it would be analyzed and processed.This project aims to provide real-time monitoring and analysis of water quality, enabling quick and effective response to any issues, thus ensuring that the water remains safe for consumption.

Fruit Ninja

This project aims to create a "Fruit Ninja" game using a FPGA (Field-Programmable Gate Array) board.In this project, the FPGA board would be used to create the game's hardware platform, while the game's software would be created using a hardware description language (HDL) such as VHDL or Verilog. The game would involve using a simple input device, such as a button or touch screen, to control the game's character and slice virtual fruits that appear on the screen.
